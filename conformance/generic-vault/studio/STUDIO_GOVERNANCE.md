# Meridian Atelier — Studio Governance

This folder (`studio/`) is the legacy governance root for Meridian Atelier, mapped to the
`{paths.legacy_governance}` token. It contains historical studio policy documents and pre-Procheiron
governance references.

## Studio Operating Principles

Meridian Atelier operates on three core principles:

1. **Material integrity** — every piece uses traceable, sustainably sourced materials. No exceptions
   for cost savings.
2. **Design transparency** — clients receive full material specifications and provenance documentation
   with each commission.
3. **Knowledge continuity** — design decisions, supplier relationships, and project state are
   documented in the shared knowledge base so any team member or agent can reconstruct context
   without tribal knowledge.

## Pre-Procheiron Governance

Before adopting Procheiron (2026-06-12), studio decisions were tracked in a shared Notion workspace.
That workspace is archived and its material facts are being migrated into the structured memory layer
as validated source records.
